分析方法論

最後更新: 2026-07-10 23:55 HKT

數據來源

  • HK 即時報價: Tencent qt.gtimg.cn (sub-1min delay，PE/PB/market_cap/52w 同時提供)
  • US 即時報價: YFinance (15-min delay)
  • 新聞: Futu Cloud (free tier, 60 req/min)
  • 歷史 bars: YFinance (HK 覆蓋不平均，新 foreign-listed 會 warn)

LLM 分析流程

  1. Fetch 報價 + 技術指標 (MA20/50/100/200 + RSI + MACD + 成交量比)
  2. Fetch 新聞 (last 5 條 + sentiment)
  3. Prompt MiniMax-M3 輸出 score_breakdown 嘅 4 個 dim (value/quality/momentum/order_flow) + 操作建議欄位 (entry/stop/target/support/resistance/confidence/trade_direction/sentiment/trend/operation_advice)
  4. Python 確定性加權：score = 0.05×value + 0.05×quality + 0.70×momentum + 0.20×order_flow（唔靠 LLM 自評分）
  5. 寫入 SQLite
  6. Rebuild static HTML + push 到 Cloudflare Pages

評分模型 — 為 day-trade 度身訂造

維度權重睇咩
估值 value_score5%PE / PB / deviation from fair value
質素 quality_score5%ROE / margin / financial health / dividend stability
動能 momentum_score70%今日方向 / MA trend / RSI / deviation（純價格動能，唔包成交量）
資金流 order_flow_score20%量比 / 大單流入 / Relative Volume vs ADV / bid-ask imbalance / 北水

Day-trade 完全偏重 momentum + order flow（合共 90%）。估值/質素只作 tiebreaker — 因為朝早 9:45 入場 11:30 出場，PE 30 定 60 對 intraday P&L 零 impact。

操作建議

每日 dashboard 約 ~93% 觀望 / ~2% 買入 / ~5% 賣出（取決於大市 regime）。LLM 綜合以下 4 個 signal 決定：

  • 🟢 買入: bullish trade_direction + score ≥ 60 + positive sentiment + 明確 setup（入場/止損/目標 齊全）
  • 🔴 賣出: bearish trade_direction + score ≤ 40 + negative sentiment + 跌穿關鍵支持位
  • 🟡 觀望: 其他（regime 唔清、setup 唔齊、score 40-60 模糊區間）

⚠️ 操作建議 唔等於單一 score threshold — LLM 會 cross-check 4-dim + sentiment + news flow + technical setup。一個 score 80 但 setup 模糊嘅股票仍然可能係觀望。

📊 Backtest 驗證 (2026-05-29 至 2026-06-29, 22 個交易日)

我哋用 historical yfinance data 跑咗 22 個交易日 × 200 隻美股 (US-only，HK 暫時冇 historical source)， 總共 4,379 個信號 × 1D/1W horizons。Hit rate 表：

信號1 日 hit rate1 週 hit rate1 日 avg move1 週 avg move
🟢 買入58.6% (n=58)64.3% (n=42)+0.91%+1.24%
🔴 賣出59.7% (n=191)48.0% (n=202)-0.40%+0.45%
🟡 觀望n/an/a+0.11%+0.72%

兩個關鍵 takeaway：

  1. 🟢 買入 multi-day 有效 — 1 週 hit rate (64.3%) > 1 日 (58.6%)，表示 buy signal 嘅 lead time work，long 倉 hold 過夜仲跟到。
  2. 🔴 賣出 day-trade only — 1 日 hit rate (59.7%) > 1 週 (48.0%)，表示 short setup mean-revert，第二日就會反彈。SELL 必須 16:00 HKT/ET 前平倉。

Implication 落 dashboard：

  • 買入 cards 標 "🟢 買入 — multi-day hold OK"
  • 賣出 cards 標 "🔴 賣出 — day-trade only, close by 4 PM"

🛡️ R:R Override 過濾 (Risk:Reward)

每個 buy setup 計算 reward / risk ratio:

  • Reward = |target_price - last| / last
  • Risk = |last - min(LLM_support, today_low)| / last

Dashboard card 會顯示 R:R badge：

  • 🟢 R:R ≥ 2.0 — setup 質量好，risk-controlled
  • 🟡 1.0 ≤ R:R < 2.0 — 普通 setup，risk/reward 1:1
  • 🔴 R:R < 1.0 — risk 大過 reward，避開

邏輯：buy setup 入場前要計清楚「贏幾多 vs 輸幾多」。R:R ≥ 2.0 表示 setup 期望值正（2:1 風險回報），否則好 setup 都係陷阱。
