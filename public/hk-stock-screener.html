點樣用 dashboard 反向篩選

除咗「由條件搵股票」(正向篩選)，好多 trader 鍾意「由結果反推條件」(反向篩選)。Leeks Terminal dashboard 嘅 detail table 支援呢個 workflow：

範例 1：搵今日 RSI < 30 + 量比 > 2 嘅超賣反彈股

  1. 去 dashboard 入面，撳「詳細表格」嘅表頭「動能」sort by 升序
  2. 再睇「估值/質素/動能/資金流」嗰一欄 (格式 v/q/m/of)，搵 m < 30 (即 RSI 超賣) 同時 of > 60 (量比放大)
  3. 符合條件嘅股，會喺當日 / 翌日有反彈機會 — 入場區間寫喺 detail page，止損位通常放 RSI < 30 嗰個底 1-2% 下面
  4. Target 預設 +3% (TP1) / +6% (TP2)，持倉 1-3 日

範例 2：搵 PE < 10 + ROE > 15% 嘅平靚正港股

  1. Sort detail table by score 倒序，搵綜合分高但 score_breakdown 顯示 v > 70 + q > 70 嘅股
  2. 呢類股通常 PE 5-10x、ROE 15-25%，係傳統價值投資心水
  3. 注意：dashboard 主力做 day-trade，呢類股未必有 🟢 買入信號 (因為 momentum 中性)，可能要等回調先入場

範例 3：搵今日大升 5%+ 但 score 只有 50 嘅「虛火」股

  1. Sort detail table by score 倒序，搵 score 40-50 但 1d return 排前面嘅股
  2. 通常係新聞 / 消息面刺激嘅單日爆發，冇基本面支持
  3. 呢類股要小心 — 大戶可能高位派貨，散戶接刀。Dashboard 會標 🟡 觀望，建議唔好入場

範例 4：搵連續 3 日都係 🟢 買入嘅「真強勢股」

  1. 去 dashboard date picker 揀連續 3 日 (例如今日、昨日、前日)
  2. 比較 3 日 detail table 入面 operation_advice = 買入嘅股 — 如果 3 日都出現，係真強勢股
  3. 呢類股可以加大倉位 (5% → 8%)，因為信號持續性高

總結：dashboard 唔止係順向 ranking，仲可以 reverse-engineer 信號 — 透過 sort、filter、cross-date 比較，你可以由 200 隻港股入面快速搵到符合你個人策略嘅 subset。詳細 4 維度框架同計算方法睇 methodology 頁。

Q: 點樣搵「PE 低 港股」？呢個篩選器有冇 value 篩選？

A: 有。「PE 低 港股」嘅定義：PE TTM < 10x (恒生指數中位數 PE 約 11-13x，< 10x 算偏低)。Leeks Terminal 港股 stock screener 用估值維度 (v_score) 處理：

  • v_score > 70 · PE 低 (通常 < 10x)、PB 低 (< 1.0x)、偏離 5 年均值低
  • v_score 40-70 · PE 中性 (10-20x)
  • v_score < 40 · PE 高 (> 20x) 或估值貴

Dashboard detail table 嘅「估值/質素/動能/資金流」欄位入面，第一個數字就係 v_score。Sort by score 倒序，再搵 v > 70 嘅股就係 PE 偏低嘅名單。常見 PE 低 港股包括內銀 (例如 939 建設銀行 PE ~4x、1398 工商銀行 PE ~4x)、內房 (部分低至 3-5x)、公用股 (2-8x)。注意：PE 低唔一定係 value trap，要 cross-check ROE (質素 q_score) 同 dividend yield 先確認。

A: 識。「RSI 超賣 港股」即 RSI14 < 30，Leeks Terminal 港股 stock screener 用動能維度 (m_score) 處理：

  • m_score < 30 · 對應 RSI14 < 30 (嚴重超賣，反彈機會大)
  • m_score 30-40 · RSI 30-40 (輕微超賣)
  • m_score 40-60 · RSI 中性 (40-60)

Q: 「港股 量比」同 Relative Volume 有咩分別？呢個篩選器點處理？

A: 「港股 量比」(volume ratio) 同 Relative Volume (RVOL) 係兩個相似但唔同嘅概念：

  • 量比 (Volume Ratio, VR) · 當日成交量 / 過去 5 日平均成交量。即時性高，適合即日鮮用。中國 A 股市場常用。
  • Relative Volume (RVOL) · 當日成交量 / 過去 20 日 (或 30 日) 平均成交量。穩定性高，適合 swing 用。Finviz / 美股市場常用。

Leeks Terminal 港股 stock screener 主要用 Relative Volume (vs 20d ADV) 計入 order_flow_score (of_score)：

  • of_score > 70 · Relative Volume ≥ 2x，有大戶入場
  • of_score 50-70 · RVOL 1.2-2x，正常偏活躍
  • of_score < 40 · RVOL < 0.8x，成交淡靜

如果你想搵「港股 量比」高嘅股 (即 RVOL ≥ 2x)，sort by score 倒序搵 of > 70 嘅股就係。Leeks Terminal 暫時唔提供即時量比 (intraday)，但 dashboard 嘅 of_score 已經反映收市後嘅 RVOL 數值，可以作為下一個交易日嘅參考。

Q: 點樣睇「港股 資金流」？呢個工具識唔識分析北水 / 大戶動向？

A: 識。Leeks Terminal 港股 stock screener 嘅 order_flow_score (of_score) 維度包含 4 個 sub-indicator：

  1. Relative Volume · 當日成交 vs 20 日 ADV (量能放大 = 大戶活動)
  2. 大單淨流入 · 透過成交股數 × 價格分大單 (≥ 100 萬 HKD) 同小單 (< 100 萬 HKD)，計淨流入比例
  3. 港股通南向資金 · 內地經滬深港通買入港股嘅淨流入 (CCASS 數據，內銀 / 內房 / 科技權重股覆蓋)
  4. Bid-ask imbalance · 買賣盤差異 (Level-2 data 提供，但 Leeks Terminal 暫時用 closing data 估算)

Dashboard detail table 入面第四個數字就係 of_score。Sort by score 倒序，再搵 of > 70 嘅股就係「港股 資金流」活躍嘅名單 — 通常係北水狂掃 / 大戶囤貨 / 業績前偷步買入嘅強勢股。注意：單看北水流入未必等於股價升，要配合 m_score > 60 先確認 momentum 同步。

Q: 點樣用呢個篩選器做「港股 估值 篩選」？value investing 適用嗎？

A: 適用但有限制。Leeks Terminal 港股 stock screener 嘅估值維度 (v_score) 同時考慮 PE / PB / deviation from 5y mean，三個 sub-indicator 並行驗證：

  • PE (TTM) · 對比恒指中位數 (~11-13x)，< 8x 算深低，> 25x 算深高
  • PB · 對比行業平均，< 0.8x 算深低 (內銀 / 內房常見)，> 5x 算深高 (科技 / 醫療常見)
  • 5 年均值 deviation · 對比該股自身 5 年 PE / PB 中位數，< -30% 算偏低

「港股 估值 篩選」做法：sort by score 倒序，搵 v > 70 + q > 70 嘅股 — 即「平 + 高 ROE」組合。注意：dashboard 主力做 day-trade，呢類股通常 score 40-60 (觀望區)，冇 🟢 買入信號。Value investing 適合 1-3 個月持倉，唔係即日鮮 timeframe。如果你做 value investing，建議用 dashboard 做 stock screening，再用自己嘅 broker 做 deep dive (睇年報 / 行業前景 / management quality)。
